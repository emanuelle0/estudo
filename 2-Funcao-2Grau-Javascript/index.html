Calculadora de Função Quadrática

Funcionalidades
O programa é capaz de calcular as raízes da função, pontos mínimos e máximos e a direção da concavidade da parábola.
Raízes
Para cacular as raízes digite abaixo os elementos A,B e C.
Caso B ou C seja 0 basta deixar em branco.
Caso o número seja negativo escreva "-" antes do número.
O programa calcula as raízes utilizando a fórmula de Bhaskara.
Pontos Máximo e Mínimos
O ponto máximo é igual (-(b/2a)) quando a < 0.
O ponto de mínima é igual a (-(delta/4a)) quando a > 0.
Parábola
A parábola terá sua concavidade para baixo quando a < 0.
A parábola terá sua concavidade para baixo quando a > 0.
Completa ou Incompleta
Se B e C diferente de zero (0), a função é Completa.
Se B ou C igual a zero (0), a função é Incompleta.
